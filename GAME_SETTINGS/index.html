    ゲーム設定システム (Game Settings System)

    Argodeフレームワークの設定管理システムです。ゲーム全体の設定（音量、テキスト速度、表示設定など）を管理し、自動的にファイルに保存・復元します。

    概要

    • 自動保存: 設定変更時に自動でファイルに保存
    • カテゴリ別管理: 音声、表示、テキストなど用途別に設定を分類
    • デフォルト値: 安全なデフォルト値を提供
    • 型安全: 設定値の型を自動判別・検証
    • 即座適用: フルスクリーンなど一部設定は即座に反映

    設定カテゴリ

    🔊 audio (音声設定)

    • master_volume: マスターボリューム (0.0-1.0)
    • bgm_volume: BGMボリューム (0.0-1.0)
    • se_volume: SE音量 (0.0-1.0)
    • voice_volume: ボイス音量 (0.0-1.0)
    • mute_audio: 音声ミュート (bool)

    🖥️ display (表示設定)

    • fullscreen: フルスクリーン表示 (bool)
    • window_size: ウィンドウサイズ (Vector2i)
    • vsync: 垂直同期 (bool)
    • show_fps: FPS表示 (bool)

    📝 text (テキスト設定)

    • text_speed: テキスト表示速度 (0.1-5.0)
    • auto_play_speed: オート再生速度 (0.5-10.0)
    • skip_read_text: 既読テキストスキップ (bool)
    • skip_unread_text: 未読テキストスキップ (bool)
    • text_size_scale: テキストサイズ倍率 (0.5-2.0)

    🎨 ui (UI設定)

    • show_text_window: テキストウィンドウ表示 (bool)
    • ui_scale: UI全体の倍率 (0.5-2.0)
    • message_alpha: メッセージウィンドウ透明度 (0.0-1.0)
    • quick_menu_enabled: クイックメニュー有効 (bool)

    ♿ accessibility (アクセシビリティ)

    • high_contrast: ハイコントラスト (bool)
    • color_blind_mode: 色覚補助モード ("none", "protanopia", "deuteranopia", "tritanopia")
    • screen_reader: スクリーンリーダー対応 (bool)
    • subtitle_enabled: 字幕表示 (bool)

    ⚙️ system (システム設定)

    • language: 言語 ("ja", "en")
    • auto_save_interval: オートセーブ間隔（秒）
    • confirm_quit: 終了時確認 (bool)
    • confirm_overwrite: 上書き保存確認 (bool)

    使用方法

    GDScriptから使用

    # SaveLoadManagerの取得
var save_manager = ArgodeSystem.get_manager("save_load")

# 設定値の取得
var master_volume = save_manager.get_master_volume()
var text_speed = save_manager.get_text_speed()
var is_fullscreen = save_manager.is_fullscreen()

# 設定値の変更（自動保存）
save_manager.set_master_volume(0.8)
save_manager.set_text_speed(1.5)
save_manager.set_fullscreen(true)

# 汎用的な設定アクセス
var custom_setting = save_manager.get_setting("ui", "message_alpha", 0.8)
save_manager.apply_setting("ui", "message_alpha", 0.9)

# 設定のリセット
save_manager.reset_settings_to_default()

# 設定のエクスポート・インポート
var settings_backup = save_manager.export_settings()
save_manager.import_settings(settings_backup)

    RGDスクリプトから使用

    # 設定コマンドを使用
settings get audio master_volume
settings set audio bgm_volume 0.7
settings set text text_speed 1.5
settings reset

# 便利コマンド
volume                    # 全音量表示
volume master 0.8         # マスター音量設定
volume bgm               # BGM音量表示

textspeed                # テキスト速度表示
textspeed 1.5            # テキスト速度設定

    ファイル構造

    user://argode_settings.cfg  # 設定ファイル（ConfigFile形式）

    設定ファイルは以下の形式で保存されます：

    [meta]
version="1.0"
save_time=1692000000
save_date="2024-08-13T15:30:00"

[audio]
master_volume=0.8
bgm_volume=0.7
se_volume=0.9

[display]
fullscreen=false
window_size="(1280, 720)"

[text]
text_speed=1.5
auto_play_speed=2.0

    イベント・シグナル

    設定システムは以下のシグナルを発行します：

    # 設定保存・読み込み
settings_saved()              # 設定保存完了
settings_loaded()             # 設定読み込み完了
settings_save_failed(error)   # 設定保存失敗
settings_load_failed(error)   # 設定読み込み失敗

    設定の適用タイミング

    • 即座適用: fullscreen, window_sizeなど一部の表示設定
    • 保存時適用: その他の設定は次回起動時に適用
    • 自動保存: 設定変更時に自動でファイルに保存

    カスタマイズ

    新しい設定カテゴリや項目を追加する場合は、SaveLoadManager.gdのdefault_settingsを編集してください：

    var default_settings = {
    "my_category": {
        "my_setting": "default_value",
        "my_number": 42,
        "my_flag": true
    }
}

    注意事項

    • 設定ファイルが破損した場合は自動的にデフォルト値に復旧
    • バージョン管理により設定形式の変更にも対応
    • 設定値は型安全に管理され、範囲外の値は自動的に補正
    • 暗号化はセーブデータのみで、設定ファイルは平文で保存
